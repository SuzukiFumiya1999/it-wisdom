# it-wisdom

## 概要
プログラミングについて質問、回答ができる質問掲示板サイトを作成しました。

## 使い方

プログラミングについて質問を写真付きで投稿したり、他人の質問に回答することができます。

役に立った質問にいいねすることができ、いいね一覧からまとめて閲覧することができます。
